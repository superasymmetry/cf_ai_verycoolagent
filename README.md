# CTA - Cloudflare Teaching Assistant #

A cool AI application built on Cloudflare Workers. 

<img width="3190" height="1706" alt="image" src="https://github.com/user-attachments/assets/1b762141-d414-491b-a9b6-ec9d64040b6f" />

To run, simply go to the deployed [link provided](https://verycoolagent.cokeymint.workers.dev/). Upload a file with your course materials or paste in your subject content, click _Generate Mataerials_, and switch to the _Flashcards_ tab or _Activities_ tab. 

<img width="3155" height="1707" alt="image" src="https://github.com/user-attachments/assets/f8184231-7cac-48e0-9f65-a4b381d8ddb5" />
<img width="3199" height="1701" alt="image" src="https://github.com/user-attachments/assets/74fe0880-2fa8-4f73-aec0-bab3bb96960d" />

You can practice with flashcards or answer quiz questions. To change the topic, simply upload or paste different content and regenerate the practice materials.
