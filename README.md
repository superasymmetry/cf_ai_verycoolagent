# CTA - Cloudflare Teaching Assistant #

A cool AI application built on Cloudflare Workers.

<img width="2352" height="265" alt="image" src="https://github.com/user-attachments/assets/91b45bb2-5b4a-4f59-a7e9-f091fd9f0cec" />

To run, simply go to the deployed [link provided](https://verycoolagent.cokeymint.workers.dev/). Upload a file with your course materials or paste in your subject content, click _Generate Materials_, and switch to the _Flashcards_ tab or _Activities_ tab.
